#SoyAnnotations

Annotations for turning soy classes into SoyData and rendering straight to templates.


##Maven:

```xml
<dependency>
  <groupId>com.github.stefanliebenberg</groupId>
  <artifactId>soy-annotations</artifactId>
  <version>1.1</version>
</dependency>
```
